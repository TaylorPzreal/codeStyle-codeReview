## Basic code style

- 编码都是utf-8
- 缩进2个空格
- 时常 format code
- ts，js，css都是单引号
- 函数不要过长
- 函数单一功能
- 一个文件不要过长，一般不要多过400行
- 多写注释 （插件Document This）
- 变量命名，不要用简写的拼音
